Building Programs to learn Solana Development

- WBA Pre-Req Program

  - Use our WBA Private Key to enroll in the WBA enrollment dApp
- Solana Starter
  - Learn how to use UMI metaplex standard
  - Mint SPL Tokens
  - Interact with Token Metadata
  - Transfer Minted Tokens
